============
Contribution
============

If you have a question, please visit the channel `#extension-builder` of
our `Slack workspace <https://typo3.slack.com>`__
or use `Stack Overflow <https://stackoverflow.com/questions/tagged/typo3>`__.

Found a bug or missing a feature? Discuss it in our Slack channel or directly
create an issue or even a pull request in our
`GitHub repository <https://github.com/FriendsOfTYPO3/extension_builder>`__.

If you want to participate in a great open source community, you should
contribute to the Extension Builder and become a team member!
